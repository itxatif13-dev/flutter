# QuickBite Restaurant App

A Flutter-based restaurant ordering application where users can select food items from a menu and automatically generate a bill.

## Features

- **Menu Browsing**: View available food items with prices and categories.
- **Cart Management**: Easily add or remove items from the shopping cart.
- **Real-time Pricing**: See the total amount update instantly as you modify your order.
- **Automatic Bill Generation**: Generate a professional order summary and bill with one click.
- **Order Confirmation**: Simple flow to finalize the order.

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Run the app using `flutter run`.

## Technologies Used

- Flutter
- Dart
- Material 3 Design
